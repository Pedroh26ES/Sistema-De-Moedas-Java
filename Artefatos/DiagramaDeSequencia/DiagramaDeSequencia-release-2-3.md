# DiagramaDeSequencia - release 2-3

Artefato das Releases 2 e 3 do Valoriza Ae.

Este arquivo funciona como indice dos diagramas de sequencia separados por caso de uso. Os diagramas seguem o modelo do gabarito, com participantes fixos, blocos numerados, mensagens numeradas, retornos tracejados e fragmentos `alt`, `loop` e `opt`.

Os diagramas foram divididos em arquivos menores porque o GitHub pode falhar ao renderizar muitos blocos Mermaid no mesmo Markdown.
